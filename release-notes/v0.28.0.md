# Phantty v0.28.0

## New

- Added a native portable updater flow with matching portable asset selection,
  staged downloads, payload validation, rollback, and an install action in the
  update prompt.
- Added portable release packages for the standard, WebView2, and no-WebView
  builds, each carrying `phantty-updater.exe` and `version.txt`.
- Added configurable key bindings, including command center visibility for the
  active shortcuts.
- Added quake mode.
- Added filtering in the SSH server list.

## Improvements

- Added download speed feedback and a confirmation prompt for cancelling active
  file transfers.
- Added a Weixin bridge stop command for remote agent sessions.

## Docs

- Documented the portable package layout and updater behavior.
