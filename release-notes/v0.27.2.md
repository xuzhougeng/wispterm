# Phantty v0.27.2

## Fixes

- Fixed SSH terminal downloads for relative paths when the interactive shell has
  not reported its current directory. Phantty now stops instead of guessing
  `~/file`, and shows a clickable setup prompt for OSC 7 current-directory
  reporting.

## Docs

- Added setup snippets for Bash, Zsh, and Fish so SSH downloads and drag-and-drop
  uploads can resolve the interactive shell's current directory.
