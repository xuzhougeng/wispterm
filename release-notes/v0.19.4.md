# Phantty v0.19.4

## Highlights

- Added `remote-session-key` so remote sessions can use a predictable browser
  pairing key instead of a random key for every Phantty process.
- Added automatic suffixing for concurrently running local instances that share
  the same fixed remote key base, using `_1`, `_2`, `_3`, and so on.

## Fixes

- Fixed the remote-key overlay dismissal logic so fixed keys of any length are
  handled correctly.

## Documentation

- Documented `remote-session-key` in the README and website config examples.
- Added and corrected the Shift-click selection design spec so the planned
  behavior matches the current implementation details.
