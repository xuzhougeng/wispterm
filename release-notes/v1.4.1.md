# WispTerm v1.4.1

> Hotfix for AI Chat auto-title requests with DeepSeek-compatible reasoning
> settings.

## Fixed

- **AI Chat auto-title requests.** Title generation no longer sends
  `reasoning_effort` when thinking is disabled, avoiding DeepSeek
  `invalid_request_error` responses during automatic rename.
- **AI Chat tab titles.** Raw API error JSON is ignored by the title cleaner, so
  failed title requests cannot rename a chat tab to an error payload.

# WispTerm v1.4.1（中文）

> 修复 AI Chat 自动标题请求里与 DeepSeek reasoning 参数相关的问题。

## 修复

- **AI Chat 自动标题请求。** 关闭 thinking 时不再发送 `reasoning_effort`，避免自动
  rename 触发 DeepSeek `invalid_request_error`。
- **AI Chat 标签标题。** 标题清理会忽略原始 API error JSON，失败的标题请求不会再把
  chat tab 改名成错误 payload。
