# Phantty v0.25.0

## New

- Restored a bundled Agent skill in release packages under
  `plugins/skills/inspect-computer-config`: it can summarize local OS, CPU,
  memory, GPU, disk, and runtime details.
- Added dedicated English and Chinese AI Agent documentation pages.

## Improvements

- Increased the maximum number of tabs from 16 to 32 and adjusted the sidebar
  numbering layout for two-digit tabs.
- Deferred panel resize work and added UI timing helpers to keep panel updates
  lighter during layout changes.
