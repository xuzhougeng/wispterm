# WispTerm v1.11.0

> This release adds scheduled AI prompt workflows, improves Agent working
> directory control, and polishes the in-app What's New panel.

## Added

- **Scheduled `/loop` prompts.** AI Copilot can now repeat a prompt at a fixed
  interval for a limited number of runs, making recurring checks and follow-up
  tasks easier to keep in one conversation.
- **Scheduled `/watch` prompts.** AI Copilot can now send a prompt at a daily or
  one-shot time, with scheduled tasks persisted across app restarts.
- **Agent working directory control.** Users can set the default Agent working
  directory from config and override it per conversation with `/cwd`.
- **Working directory sandboxing.** Agent command approval now understands the
  effective working directory so trusted commands can stay inside the selected
  workspace while protected paths still require confirmation.

## Changed

- **Agent tool context.** AI tool prompts now include the effective working
  directory so command execution and file operations line up with the selected
  workspace.
- **What's New readability.** The in-app What's New panel now has a denser
  release layout with highlights, clearer section structure, and a calmer
  footer action row.

## Fixed

- **What's New footer layout.** The View on GitHub action is now positioned on
  the left side of the footer, with the OK action kept separate on the right.
- **What's New text wrapping.** Long release lines are measured more carefully
  so the panel avoids awkward button and content overlap.

## Known limitations

- **macOS Intel artifact.** The automatic macOS release ships Apple Silicon
  (`aarch64`) only. The Intel `x86_64` DMG remains a manual build and is not
  included in this release.

# WispTerm v1.11.0（中文）

> 本版本新增 AI 定时提示工作流，改进 Agent 工作目录控制，并继续打磨应用内 What's New
> 面板。

## 新增

- **`/loop` 定时提示。** AI Copilot 现在可以按固定间隔重复发送提示，并限制执行次数，
  便于在同一个会话里处理周期性检查和后续任务。
- **`/watch` 定时提示。** AI Copilot 现在可以在每日固定时间或一次性时间点发送提示，
  计划任务会在应用重启后继续保留。
- **Agent 工作目录控制。** 用户可以通过配置设置默认 Agent 工作目录，也可以在单个会话
  中用 `/cwd` 覆盖。
- **工作目录沙箱。** Agent 命令审批现在会理解实际工作目录，让可信命令保持在选定工作
  空间内执行，同时对受保护路径继续要求确认。

## 调整

- **Agent 工具上下文。** AI 工具提示现在会包含实际工作目录，让命令执行和文件操作与
  当前选定工作空间保持一致。
- **What's New 可读性。** 应用内 What's New 面板现在使用更紧凑的发布布局，包含
  highlights、更清晰的章节结构，以及更稳定的底部操作区。

## 修复

- **What's New 底部布局。** View on GitHub 操作现在放在底部左侧，OK 操作独立保留在
  右侧。
- **What's New 文本换行。** 长 release 行现在会更仔细地测量，避免按钮和正文出现别扭的
  重叠。

## 已知限制

- **macOS Intel 产物。** 自动 macOS 发布只包含 Apple Silicon（`aarch64`）版本。Intel
  `x86_64` DMG 仍然是手动构建，本次发布不包含它。
