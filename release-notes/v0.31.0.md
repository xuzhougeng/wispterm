# Phantty v0.31.0

## Changed

- Reworked how updates are delivered. When a newer release is available, the
  update reminder now downloads the matching portable `.zip` to your Downloads
  folder and opens it in Explorer with the file selected. Unzip it over your
  existing install to update.
- Removed the bundled `phantty-updater.exe` and the in-place self-replace
  flow. Portable zips no longer contain the updater helper.
