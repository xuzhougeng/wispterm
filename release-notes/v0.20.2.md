# Phantty v0.20.2

## Fixes

- Made the terminal scrollbar remain visible when scrollback exists and accept
  direct drag/click input without requiring a mouse wheel gesture first.
- Fixed Weixin iLink bridge AI follow-up replies so slow AI Agent tasks still
  send the final assistant message after the fixed progress checkpoints have
  already reported that processing is still in progress.

## Improvements

- Added desktop version reporting through `phantty.exe --version` / `-v` and a
  command center `Version` action.
- Added the Phantty Remote web version label on both login and console shells.
- Adjusted Weixin iLink bridge progress confirmations to reply immediately,
  then at 10, 30, 60, and 120 seconds while work is still running.

## Documentation

- Updated the Remote Weixin bridge documentation to describe progress
  checkpoints plus final-reply delivery on later AI Chat snapshot updates.
- Documented the release version synchronization checklist in `AGENTS.md`.
