# WispTerm v1.9.0

> This release adds a private file-access guard for AI Agent shell tools,
> direct panel focus by number, and a macOS CJK fallback-font fix.

## Added

- **Private Agent file-access rules.** WispTerm now loads
  `~/.config/wispterm/agent-access.local` as a machine-local allow/deny file
  for Agent shell access. Denied paths force approval even in full permission
  mode, while read-only commands confined to allowed roots can skip confirmation.
- **Secure-by-default Agent deny list.** Even without a private rules file, the
  Agent gate protects common sensitive paths and names such as SSH, cloud,
  Kubernetes, GitHub CLI, Docker credentials, `.env`, `*.pem`, and `*.key`.
- **Panel focus by number.** Use `Ctrl+1` through `Ctrl+9` on Windows/Linux or
  `Cmd+1` through `Cmd+9` on macOS to focus split panels in screen reading
  order, top-left to bottom-right.

## Fixed

- **Protected attachment sends.** `weixin_send_attachment` now goes through the
  same Agent file-access deny rules before uploading a file.
- **macOS CJK fallback fonts.** macOS now loads CJK fallback fonts from memory
  so Chinese/Japanese/Korean glyphs no longer render as tofu when the fallback
  needs to bridge through Core Text.

## Known limitations

- **Agent file-access enforcement is heuristic.** It gates ordinary shell
  command text and intentionally biases toward extra prompts for protected
  paths; it is not an OS sandbox.
- **macOS Intel DMG is manual.** The Intel DMG is still built by the on-demand
  workflow after the tag-triggered release creates the GitHub release.

# WispTerm v1.9.0（中文）

> 本版本新增 AI Agent 私有文件访问规则、按编号直接聚焦面板，并修复 macOS CJK fallback
> 字体问题。

## 新增

- **Agent 私有文件访问规则。** WispTerm 现在会读取
  `~/.config/wispterm/agent-access.local` 作为本机私有 allow/deny 规则文件。
  命中 deny 的路径即使在 full 权限模式下也会强制确认；只读命令如果完全限制在 allow
  根目录内，则可以跳过确认。
- **Agent 默认敏感路径保护。** 即使没有私有规则文件，Agent gate 也会默认保护 SSH、
  云服务、Kubernetes、GitHub CLI、Docker 凭据、`.env`、`*.pem` 和 `*.key`
  等常见敏感路径或文件名。
- **按编号聚焦面板。** Windows/Linux 使用 `Ctrl+1` 到 `Ctrl+9`，macOS 使用
  `Cmd+1` 到 `Cmd+9`，即可按屏幕阅读顺序（左上到右下）聚焦当前标签页内的分屏面板。

## 修复

- **受保护附件发送。** `weixin_send_attachment` 现在会先经过同一套 Agent 文件访问
  deny 规则，再上传文件。
- **macOS CJK fallback 字体。** macOS 现在从内存加载 CJK fallback 字体，避免经过
  Core Text fallback 时中日韩字符显示为 tofu 方块。

## 已知限制

- **Agent 文件访问规则是启发式保护。** 它检查普通 shell 命令文本，并会对受保护路径
  偏向额外确认；它不是 OS 级 sandbox。
- **macOS Intel DMG 需要手动触发。** Intel DMG 仍会在 tag 触发的 release 创建后，由按需
  工作流单独构建并上传。
