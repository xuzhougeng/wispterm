# WispTerm v1.20.0

> This release adds native PDF previews, lets image/Markdown/PDF preview panes
> stack per kind, and introduces a Copilot subagent research path with safer
> WeChat and config-reload behavior.

## Added

- **Native PDF preview panes.** `.pdf` files now open as preview panes with
  page indicators, page-specific failure messages, async page-flip jobs,
  `PageUp`/`PageDown` navigation, and raster zoom/pan.
- **Platform PDF rasterizers.** PDF preview dispatches to Windows WinRT,
  macOS CGPDF, and Linux Poppler backends, with unsupported-platform stubs kept
  behind the platform interface.
- **Per-kind stacked previews.** Image, Markdown, and PDF previews can coexist
  in the preview column. Opening a preview reuses an existing pane of the same
  kind when possible and stacks new kinds predictably.
- **Copilot subagent research.** The agent prompt and tool schema can delegate
  research-style tasks to a nested subagent loop, with `ai-subagent-profile`
  selecting the credential profile and test seams for model/profile override
  behavior.

## Changed

- **Focused preview close behavior is clearer.** `Ctrl+Shift+W` closes the
  focused pane, so a focused preview closes without taking focus away from the
  split tree first.
- **Preview documentation is broader.** The English and Chinese README, wiki,
  FAQ, file explorer docs, and embedded configuration docs now describe PDF
  preview and stacked preview panes.
- **WeChat direct-send guidance is clearer.** The toast now says when a restart
  is required for the updated direct-send setting to take effect.

## Fixed

- **Config reload lifetime safety.** Config reload no longer aliases
  config-owned strings into long-lived font globals.
- **PDF/image zoom bounds.** Precise macOS scroll input can no longer drive
  image or PDF preview zoom to unusable extremes.
- **Image preview drag-pan.** Left-drag panning is restored for image preview
  panes and covered by a compile-time mouse-wiring guard.
- **WeChat multiline prompts.** A multiline WeChat message is submitted as one
  prompt instead of being split into separate prompts at newlines.
- **WeChat manual stop handling.** `Stopped` is treated as a terminal status so
  a manual Copilot stop ends the follow-up flow cleanly.
- **PDF preview edge cases.** PDF preview code-review findings were addressed
  around page state, raster dispatch, and failure handling.

## Known limitations

- **No macOS x86_64 DMG for this release.** v1.20.0 intentionally skips the
  manual Intel macOS workflow and publishes the Apple Silicon macOS DMG only.
- **Linux is experimental.** The Linux AppImage is published for community
  testing and is not yet considered stable.

# WispTerm v1.20.0（中文）

> 本版本新增原生 PDF preview，让 image/Markdown/PDF preview pane 可以按类型堆叠，
> 并加入 Copilot subagent research 路径，同时强化 WeChat 和 config reload 的边界行为。

## 新增

- **原生 PDF preview pane。** `.pdf` 文件现在会以 preview pane 打开，支持页码提示、
  单页失败信息、异步翻页任务、`PageUp`/`PageDown` 翻页，以及 raster zoom/pan。
- **平台 PDF rasterizer。** PDF preview 会分发到 Windows WinRT、macOS CGPDF 和
  Linux Poppler 后端，不支持的平台仍通过 platform interface 后的 stub 保持隔离。
- **按类型堆叠 preview。** Image、Markdown 和 PDF preview 可以共存在 preview
  column 中。打开 preview 时会尽量复用同类型 pane，新类型会按稳定位置堆叠。
- **Copilot subagent research。** Agent prompt 和 tool schema 现在可以把 research
  类任务委派给嵌套 subagent loop，并通过 `ai-subagent-profile` 选择凭据 profile；
  model/profile override 行为也有测试 seam 覆盖。

## 调整

- **Focused preview 关闭行为更明确。** `Ctrl+Shift+W` 现在关闭当前 focused pane，
  因此焦点在 preview 上时会直接关闭该 preview，而不是先把焦点移回 split tree。
- **Preview 文档覆盖更完整。** 英文/中文 README、wiki、FAQ、File Explorer 文档和内置
  configuration docs 现在都说明了 PDF preview 与 stacked preview pane。
- **WeChat direct-send 提示更清楚。** Toast 现在会说明 direct-send 设置需要重启后生效。

## 修复

- **Config reload 生命周期安全。** Config reload 不再把 config 拥有的字符串别名写入长生命周期
  font globals。
- **PDF/image zoom 边界。** macOS precise scroll input 不再能把 image 或 PDF preview
  zoom 推到不可用的极端值。
- **Image preview 拖拽平移。** Image preview pane 的左键拖拽平移已恢复，并新增 compile-time
  mouse-wiring guard 覆盖。
- **WeChat 多行 prompt。** 多行 WeChat 消息现在会作为一个 prompt 提交，而不是按换行拆成多个 prompt。
- **WeChat 手动停止处理。** `Stopped` 现在被视为终态，手动停止 Copilot 后 follow-up 流程会干净结束。
- **PDF preview 边界情况。** 针对 page state、raster dispatch 和失败处理的 code-review 问题已修复。

## 已知限制

- **本次不发布 macOS x86_64 DMG。** v1.20.0 会跳过手动 Intel macOS workflow，只发布
  Apple Silicon macOS DMG。
- **Linux 仍为实验性质。** Linux AppImage 用于社区测试，尚不视为稳定版本。
