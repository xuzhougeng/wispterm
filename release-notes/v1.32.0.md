# WispTerm v1.32.0

> This release adds the Windows native D3D11 feedback package, keeps OpenGL as
> the default Windows renderer, retires the no-WebView zip, and includes the
> recent command, update, screenshot, remote-file, Weixin, and module-boundary
> improvements.

## Added

- **Windows native D3D11 feedback package.** Windows releases now include
  `wispterm-windows-portable-native-d3d11-*.zip` for users who want to test the
  native renderer while the default Windows package remains OpenGL.
- **Command snippets (#368).** The command center can now run user-defined
  command snippets.
- **macOS one-click update (#411).** macOS builds now support the immediate
  update flow from the update prompt.
- **Active-tab UI screenshots (#410).** Agent tooling can capture the active tab
  UI state for visual debugging.
- **Remote file edit helper (#415).** Remote file tooling now includes an edit
  helper for agent workflows.
- **Weixin progress slash commands (#414).** Weixin agent progress can now be
  controlled through slash commands.
- **Right-click tab rename (#407).** Tabs can be renamed from the right-click
  tab menu.

## Changed

- **Windows no-WebView package retired.** Releases no longer publish a separate
  no-WebView zip; use the default OpenGL package unless you need the compat or
  native D3D11 feedback package.
- **Agent and assistant modules were reorganized.** Agent tools, assistant
  conversation/sidebar modules, SSH helpers, preview, command, skill, research,
  port-forwarding, and related renderers moved into clearer feature folders.
- **Assistant token speed metrics (#406).** Assistant output now reports token
  speed metrics.
- **Source layout guards were expanded.** CI now enforces more of the new
  feature-folder ownership boundaries.
- **UI screenshots are capped (#413).** Screenshot capture now limits output
  resolution to keep artifacts smaller.

## Fixed

- **Remote file tool targets (#412).** Remote file tools now resolve the intended
  target more reliably.
- **Child process output draining.** Process capture now drains final pipe bytes
  after child exit.
- **macOS e2e coverage (#385).** Real app tests now cover tabs, splits, paste,
  click, resize, and OSC title flows.

## Known limitations

- **Windows native D3D11 is feedback-only.** Use the default OpenGL package if
  the native package shows a black window, crash, missing UI, resize failure,
  RDP issue, or multi-monitor/DPI issue. Please include diagnostics, GPU/driver,
  Windows version, and display/session topology when reporting native-renderer
  issues.
- **Linux is experimental.** The Linux AppImage is published for community
  testing and is not yet considered stable.
- **macOS remote access remains early.** The macOS relay transport is available
  but still needs more end-to-end field testing.

# WispTerm v1.32.0（中文）

> 本版本新增 Windows native D3D11 反馈包，Windows 默认 renderer 仍为 OpenGL，
> 同时退役 no-WebView zip，并包含近期 command、更新、截图、远程文件、微信和
> 模块边界整理改进。

## 新增

- **Windows native D3D11 反馈包。** Windows release 现在包含
  `wispterm-windows-portable-native-d3d11-*.zip`，用于测试 native renderer；
  默认 Windows 包仍然使用 OpenGL。
- **Command snippets（#368）。** 命令中心现在可以运行用户自定义命令片段。
- **macOS 一键更新（#411）。** macOS 构建现在支持从更新提示里立即执行更新。
- **Active-tab UI 截图（#410）。** Agent 工具现在可以捕获当前 active tab 的 UI 状态，
  用于可视化调试。
- **Remote file edit helper（#415）。** 远程文件工具现在包含面向 agent workflow 的编辑 helper。
- **Weixin progress slash commands（#414）。** Weixin agent progress 现在可以通过 slash
  commands 控制。
- **右键重命名 tab（#407）。** 现在可以从 tab 右键菜单重命名 tab。

## 调整

- **Windows no-WebView 包退役。** Release 不再发布单独的 no-WebView zip；除非需要
  compat 或 native D3D11 反馈包，否则使用默认 OpenGL 包。
- **Agent 和 assistant 模块继续整理。** Agent tools、assistant conversation/sidebar、
  SSH helpers、preview、command、skill、research、port-forwarding 以及相关 renderer
  已迁移到更清晰的 feature folders。
- **Assistant token speed metrics（#406）。** Assistant 输出现在会报告 token speed metrics。
- **Source layout guards 扩展。** CI 现在会继续约束新的 feature-folder ownership 边界。
- **UI screenshot 分辨率限制（#413）。** 截图捕获现在会限制输出分辨率，避免 artifact 过大。

## 修复

- **Remote file tool targets（#412）。** 远程文件工具现在更可靠地解析目标。
- **子进程输出收尾读取。** Process capture 现在会在 child exit 后继续 drain 最后的 pipe bytes。
- **macOS e2e 覆盖（#385）。** 真实应用测试现在覆盖 tabs、splits、paste、click、resize 和
  OSC title 流程。

## 已知限制

- **Windows native D3D11 仅作为反馈包。** 如果 native 包出现黑屏、崩溃、UI 缺失、
  resize 失败、RDP 问题或多屏/DPI 问题，请切回默认 OpenGL 包。反馈 native renderer
  问题时，请附上诊断、GPU/驱动、Windows 版本和显示器/会话拓扑。
- **Linux 仍为实验性质。** Linux AppImage 用于社区测试，尚不视为稳定版本。
- **macOS 远程访问仍为早期功能。** macOS 中继传输已经可用，但仍需要更多端到端实际验证。
