# Phantty v0.17.0

## Highlights

- Added opt-in tab session persistence: enable
  `restore-tabs-on-startup = true` in your config and Phantty will save
  the current tab/split layout when you close the last window, and rebuild
  it on the next launch. SSH targets reconnect using the stored
  host/user/port, but always re-prompt for credentials — passwords are
  never written to disk. Local shells reopen at their last reported cwd.

## Other changes

- Fixed a crash that could happen after closing a split, caused by stale
  split hit-test/render caches still pointing at the removed surface.
- Reverted the spatial split focus shortcut to `Alt` + arrow keys (was
  briefly changed to `Ctrl+Shift` + arrows in v0.16).
