# Phantty v0.20.0

## Highlights

- Added the Phantty Remote Weixin iLink bridge, including server-side binding,
  QR login polling, session routing, and browser controls in the remote console.
- Added Weixin bridge settings and persistence so the remote web console can
  configure app credentials and route incoming messages to the selected session.
- Improved remote protocol support for larger browser and agent messages.

## Fixes

- Fixed fragmented remote WebSocket messages so large web-side agent messages no
  longer disconnect the desktop relay client.
- Fixed Weixin bridge save, bind, stale poller, and teardown races in the remote
  server.
- Fixed mobile remote tabs on short phone screens by allowing the tab list to
  scroll within the available viewport.

## Documentation

- Added Remote Weixin bridge setup documentation and implementation notes.
- Added design guidance for the current remote UI mobile viewport constraints.
