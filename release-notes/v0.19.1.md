# Phantty v0.19.1

## Highlights

- AI Chat is easier to control while an agent is working: message bubbles now
  include a copy button, transcript/input selection supports `Ctrl+A` and
  `Ctrl+C`, and a visible stop button can cancel an in-flight request.
- Terminal text selection now supports Word-style `Shift-click`: click once to
  set an anchor, then `Shift-click` an endpoint to select the range between
  them.

## Fixes

- AI tool snapshots now capture the active terminal list from the UI thread so
  agent tools can see existing terminal and SSH sessions correctly.
- Cancellation suppresses late AI results and stops running PowerShell tool
  commands when possible.

## Documentation

- Documented the new terminal `Shift-click` selection behavior in the shortcut
  reference and startup overlay.
