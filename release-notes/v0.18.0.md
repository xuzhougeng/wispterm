# Phantty v0.18.0

## Highlights

- Added AI Chat tabs with DeepSeek/OpenAI-compatible profiles, native Zig HTTP
transport, configurable thinking/reasoning/stream options, and live streaming
responses in the chat UI.
- Moved the file explorer to the left sidebar and added text preview on double
click, making local/SSH browsing fit better with tab and split workflows.

## AI Chat

- Added an AI Chat profile form with saved profile support, API key handling,
model/base URL/system prompt fields, and DeepSeek defaults.
- Replaced the initial `curl.exe` helper transport with `std.http.Client`.
- Added configurable `thinking`, `reasoning_effort`, and `stream` request
fields matching DeepSeek's chat completion API.
- Implemented SSE stream parsing that appends `delta.content` and
`delta.reasoning_content` directly into the active AI message as chunks
arrive.
- Improved AI Chat rendering with full-width layout, bottom anchoring, clipped
transcript rendering, reasoning block backgrounds, and proper IME candidate
positioning at the chat input field.

## File Explorer and Preview

- Moved the file explorer panel to the left side with updated shortcut text and
sidebar behavior.
- Added double-click text preview for file explorer entries.
- Hardened Ctrl-click preview token parsing so trailing punctuation is not
included in preview paths.

## Fixes

- Added close confirmation safeguards for cases that would close the last tab
or window.
