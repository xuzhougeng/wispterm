# Phantty v0.21.0

## Improvements

- Added persistent AI/Agent conversation history with stable `session_id`
  tracking, so old chats can be restored and continued instead of starting from
  scratch every time.
- Added an `AGENT History` mode to the existing left sidebar: `Ctrl+Shift+E`
  now shows conversation history on AI/Agent tabs, while terminal tabs keep the
  existing file explorer.
- Added history-aware tab reuse so reopening a saved AI conversation switches to
  an already-open matching tab instead of duplicating the same session.
- Improved left-sidebar retargeting so the file explorer stays in sync with the
  currently focused terminal surface, including local, WSL, and SSH splits.
- Improved history-row rendering and caching so long UTF-8 titles and models do
  not break display text when truncated for the sidebar.

## Fixes

- Fixed AI-history sidebar focus and mode transitions so switching tabs, moving
  between splits, or activating a saved conversation no longer leaves the
  sidebar stuck on stale file/history state.
- Fixed unnecessary file-explorer resets when the focused terminal split changes
  but the effective target directory/connection is unchanged.

## Documentation

- Added the agent-history design spec and implementation plan under
  `docs/superpowers/`.
