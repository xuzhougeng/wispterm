# Phantty v0.23.0

## Improvements

- Improved Phantty Remote mobile console behavior: selecting a sidebar item now opens the target conversation and collapses the sidebar, mobile zoom overlays stay anchored, and mobile input mode is simplified to `View` / `Edit` with a dedicated `Key` shortcut keyboard toggle.
- Added Phantty Remote connection status lights with red / green / yellow states and click-to-view latency in milliseconds. The high-latency threshold is 500 ms.
- Simplified remote AI chat snapshots so the web UI keeps complete question and answer content while keeping intermediate tool-call display compact.
- Added deletable Agent History entries from the desktop history UI and command-center history picker.

## Fixes

- Fixed remote AI Agent synchronization so the browser sees the latest AI chat messages.
- Fixed Weixin bridge replies so final AI answers are sent after tool/progress messages instead of only showing intermediate status.
