# Phantty v0.20.3

## Improvements

- Added richer AI Chat composer editing with left/right/home/end/delete support,
  multiline cursor movement, a taller auto-growing input area, and an
  interactive input scrollbar for long prompts.
- Agent tool commands now run as hidden background child processes where
  possible, so local `cmd.exe` / PowerShell helper windows do not flash while
  AI Agent tools are working.
- Completed AI and Agent responses now show elapsed time, plus token usage when
  the provider returns OpenAI-compatible `usage` fields.
- Added terminal mouse clipboard options via `copy-on-select` and
  `right-click-action`, including `copy-or-paste` behavior for Xshell-style
  workflows.
- Added `ssh-legacy-algorithms = true` for older bastions or servers that still
  require `ssh-rsa`, `ssh-dss`, older Diffie-Hellman KEX, or CBC ciphers.

## Fixes

- Fixed the split-pane scrollbar hit testing and dragging behavior so custom
  scrollbars work reliably before mouse-wheel interaction.

## Documentation

- Documented the custom-drawn UI rationale in `README.md`.
- Added GitHub Pages FAQ entries for enabling copy-on-select and right-click
  paste.
