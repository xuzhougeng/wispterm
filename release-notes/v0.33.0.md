# Phantty v0.33.0

## Added

- Added the first native macOS application path: Metal rendering, Cocoa window
  backend wiring, platform service implementations, macOS packaging notes, and
  smoke tests for the native UI path.
- Added POSIX PTY and process support groundwork, including fork/exec, ioctl
  sizing, process reaping, and macOS PTY constants.
- Added persistent SSH browser tunnels so embedded browser sessions can keep
  using SSH-backed local forwarding across panel activity.
- Added a user-facing render debug mode for investigating high-DPI rendering
  glitches.

## Fixed

- Stabilized synchronized terminal output rendering and IME caret placement,
  reducing flicker and caret drift in TUI-heavy sessions.
- Tightened POSIX child-process failure handling so fork children exit through
  the low-level process exit path.

## Internal

- Continued the renderer backend split by routing remaining OpenGL-heavy
  renderer paths through `gpu` and `ui_pipeline` primitives, adding regression
  guards that keep raw GL use contained.
- Added Metal backend primitives and build coverage so the project can
  cross-compile toward native macOS targets.
